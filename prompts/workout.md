React + Vite 프론트엔드와 FastAPI 백엔드로 운동 기록 앱을 만들어줘.
프론트엔드 스캐폴드(main.jsx, index.css, vite.config.js)는 client/ 디렉토리에 이미 세팅되어 있어.

요구사항:
- 운동 기록을 추가할 수 있어야 해 (운동 이름, 세트 수, 반복 수, 무게, 날짜)
- 기록을 수정하고 삭제할 수 있어야 해
- 날짜별로 기록을 목록으로 보여줘야 해
- 운동별 총 볼륨(세트 x 반복 x 무게)을 계산해서 보여줘야 해
- 백엔드는 FastAPI + SQLite로 구현해
- 서버는 port 8000에서 실행해
